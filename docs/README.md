# Documentation

This directory contains additional documentation for cc-daemon.

## Structure

- `gui/` - GUI development documentation
  - GUI implementation details and future improvements

- `development/` - Development process documentation
  - Iteration plans and implementation summaries
  - Debugging guides and test results

- `research/` - Research findings
  - Initial research and architecture decisions

## Working Documentation

For active development work, see `workspace/analysis/` which contains detailed analysis files for each module.
